# PalindroneCheckerApp

##  Objective
To build a console-based **PalindromeChecker App** in Java that validates whether a string is a palindrome, while reinforcing programming fundamentals and data structure concepts.

##  Flow of Project
1. **UC1 – Application Entry & Welcome Message**
    - Program starts with `main()`.
    - Displays app name and version.
    - Establishes startup flow.

2. **UC2 – User Input & Palindrome Validation**
    - Accepts string input from user.
    - Checks if it is a palindrome.
    - Displays result.

3. **UC3 – Extended Conditions**
    - Offers strict vs. relaxed validation rules.
    - Processes accordingly.

4. **UC4 – Multiple Checks & Exit Flow**
    - Allows repeated checks.
    - Provides exit option.

##  Use Cases Summary
- **UC1:** Startup & Welcome.
- **UC2:** Palindrome check.
- **UC3:** Rule-based validation.
- **UC4:** Multiple checks + graceful exit.
